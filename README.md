This project is a real-time Computer Vision application that detects human faces via a webcam and analyzes their expressions. Built with Python, it utilizes the DeepFace framework for high-accuracy emotion classification (such as happy, sad, angry, surprise, neutral) and leverages OpenCV for hardware interfacing and on-screen visualization.

This script demonstrates the practical implementation of pre-trained deep learning models for continuous, live visual data analysis.

⚙️ Technologies & Libraries Used
Language: Python

Core AI Framework: DeepFace (Facial Recognition & Attribute Analysis)

Image Processing: OpenCV (cv2)

Backend: TensorFlow / Keras
How to Run the Project
Clone this repository to your local machine: git clone https://github.com/ezcool2000/Facial-Emotion-Recognition.git
Install the required dependencies: pip install -r requirements.txt
Run the Python script: python main.py
(Note: Press 'q' to quit the application and close the webcam feed.)
